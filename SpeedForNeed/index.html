Speed for Need (WIP)

Ein 3-Spieler 2D Rennspiel. Gemacht zusammen mit meinem Cousin in Unity.

Download (für Windows)  Auf GitHub anzeigen 

Speed for Need ist ein 3-Spieler 2D Rennspiel. Man kann das rote Auto mit WASD steuern, das blaue mit den Pfeiltasten und das grüne mit IJKL.
